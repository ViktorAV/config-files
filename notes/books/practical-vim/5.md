# Глава 5. Режим командной строки.
Редактор Vim ведет свою родословную от редактора vi, в котором была реализована
модальная парадигма редактирования.
Редактор Vi, в свою очередь, ведет свою родословную от строчного редактора __Ex__,
от которого по наследству достались команды __Ex__.
Режим командой строки позволяет вводить команды редактора __Ex__, шаблоны поиска или выражения.

Справка по командам Ex `:h ex-cmd-index`.

Стиль автодополнения Vim. `:h wildmode`.
```vim
set wildmenu
set wildmode=full, longest,list
```

__Важно__
- Команды, выполняемые в командной строке, называют командами редактора Ex.
- Команды Ex (`:`) имеют более широкую область действия.
- По умолчанию Vim запоминает до 20 команд, увеличить этот размер можно так `set history=200`.


## Команды
- `:write | %p` Оператор `|` связывает две команды.
- `q:` Отобразить историю введенных команд.
- `:` Вход в режим командной строки.
- `/` Вход в режим командной строки, шаблон поиска.
- `:!ls` Запускает команду шела(оболочки).
- `<C-r>=` Вход в режим командной строки, регистр.
- `:1` Переходит на первую строку файла, `:$` конец.
- `:{start}, {end}` Выбрать диапазон строк.
- `:/<html>/, /<\/html>/` Выбрать по шаблону.
- `:12+1, 12-1` Выбрать от и до со смещением.
- `:.` Выбрать текущую строку.
- `:%` Выбрать все строки файла.
- `'<,'>` Выбрать выделенный участок текста.
- `:[range]delete[x]` Удалит указанные строки в регистр `x`.
- `:[range]yank[x]` Скопирует указанные строки в регистр `x`.
- `:[line]put[x]` Вставит текст из регистра `x` после указанной строки.
- `:[range]copy{address}` Скопирует указанные строки, ниже строки с номером `address`.
- `:[range]move{address}` Переместит указанные строки, ниже строки с номером `address`.
- `:[range]join` Объеденит указанные строки.
- `:[range]normal{commands}` Выполнит команду `commands` для каждой указанной строки.
- `:[range]substitute/{pattern}/{string}/[flags]` Заменит вхождения `pattern`
  Последовательностью `string` в каждой указанной строке.
- `:[range]global/{pattern}/[cmd]` Выполнит команду `cmd` для
   указанных строк со вхождением `pattern`.
- `:p :print` Отображает строку в статус баре.
- `:write` Сохранить изменения в файле.
- `:edit` Открыть файл.
- `:tabnew` Создать новую вкладку.
- `:split` Разделить окно.
- `:prev :next` Список аргументов.
- `:bp :bprev :bn :bnext` Перейти к буферу.
- `:ls` Вывести список буферов.
- `:co :copy` Копирует строку.
- `:m :move` Перемещает строку.
- `:normal` Выполняет команду командного режима для диапазана строк. (`:1,5normal .`)
  `:%normal A;` Добавить `;` ко всем строкам, `:%normal i//` закоментировать весь файл js.
- `:t$` Скопировать текущую строку и вставить в конец файла.
- `:delete`
- `:yank`
- `:put`
- `:s :substitute` Замена текста по шаблону.
- `:global`
- `:args` Список открытых файлов?
- `:first :next` Перейти к файлам.
- `:source` Выполнить сценарий.
- `:read` Переадресует в буфер, для удобочитаемости.
- `<C-d>` Автодополнение в Vim. Или список доступных вариантов.
- `:set` Установить значение параметру.
- `:help` Отобразить справку по команде.
- `:shell` Запускает интерактивный сеанс командной оболочки.
