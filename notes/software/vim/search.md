# Поиск
- __/__
  Выполняет поиск по шаблону, поддерживает регулярные выражения.
- __n N__
  Переходит к следующему или предыдущему результату поиска.
- __?__
  Выполняет поиск назад.
- *
  Выполняет поиск слова под курсором.
- __f__`char`
  Переходит к `char`.
- __F__`char`
  Переходит к `char` назад.
